# Shell Scripting
- Shebang is the first line in a script file
- It tells the system which interpreter should run the script
- **Shebang (#!)**
```bash
#!/bin/bash       # Run script using bash
#!/bin/sh         # Use the default shell (sh)
#!/bin/zsh        # Use zsh
#!/usr/bin/python3  # Run with Python
```


## To run
```bash
#!/bin/bash

chmod +x hello.sh

./hello.sh


#alternative
bash hello.sh

```
